# Support

Need help with a [beardwhocodes](https://github.com/beardwhocodes) project?

- **Questions & bugs** → open an issue on the relevant repository (use the issue
  templates). Please search existing issues first.
- **Security** → see the project's `SECURITY.md`; report privately, not as a
  public issue.
- **Elsewhere** → [beardwho.codes](https://beardwho.codes) ·
  [@beardwhocodes](https://x.com/beardwhocodes)
